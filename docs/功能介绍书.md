# 博客后台管理系统 - 功能介绍书

## 1. 产品简介

博客后台管理系统（Blog-Front-MS）是一个基于 **Vue 3 + TypeScript + Element Plus** 技术栈构建的现代化博客管理平台前端项目。系统采用前后端分离架构，专注于为博主提供高效、便捷的内容管理体验。

---

## 2. 功能模块详解

### 2.1 登录认证

**功能描述**: 系统入口，用户通过账号密码登录获取访问权限。

**核心能力**:
- 用户名/密码表单登录，支持回车快捷提交
- 表单实时校验（非空校验、格式验证）
- 登录状态持久化（Token + UserInfo 存储至 localStorage）
- 路由守卫：未登录自动跳转登录页，已登录禁止重复登录
- Token 过期/失效时自动登出
- 渐变背景 + 居中卡片式 UI 设计

**技术要点**:
- `vue-router` 的 `beforeEach` 导航守卫实现权限控制
- `pinia-plugin-persistedstate` 实现状态持久化
- `ElForm` 组件实现表单校验

---

### 2.2 仪表盘

**功能描述**: 数据可视化首页，一页掌握博客运营全貌。

**核心能力**:
- **统计卡片组**: 文章总数、评论总数、累计访问量、注册用户数 四大核心指标
- **访问趋势图**: ECharts 折线图展示近 6 个月文章访问量趋势（带面积填充）
- **分类占比图**: ECharts 环形饼图展示各分类文章数量占比
- **最近文章列表**: 展示最新发布的 5 篇文章（标题、分类、阅读量、状态、时间）

**技术要点**:
- ECharts 响应式图表（监听 window.resize 自适应）
- 组件卸载时调用 `dispose()` 防止内存泄漏
- `el-row` / `el-col` 栅格系统响应式布局

---

### 2.3 文章管理

**功能描述**: 系统最核心模块，提供文章的全生命周期管理。

#### 2.3.1 文章列表页

| 功能 | 说明 |
|------|------|
| 多条件搜索 | 按标题关键词搜索 |
| 分类筛选 | 下拉选择分类过滤 |
| 状态筛选 | 已发布 / 草稿 |
| 排序 | 按阅读量排序 |
| 分页 | 支持切换每页条数 (10/20/50) |
| 新建 | 跳转编辑页创建新文章 |
| 编辑 | 跳转编辑页修改文章 |
| 删除 | 二次确认后删除 |

#### 2.3.2 文章编辑页

| 功能 | 说明 |
|------|------|
| 标题输入 | 必填，限制长度 |
| 摘要编辑 | 多行文本，用于 SEO 和列表预览 |
| 内容编辑 | Markdown 编辑器区域（预留接口） |
| 封面上传 | 图片上传与预览 |
| 分类选择 | 单选下拉框 |
| 标签选择 | 多选下拉框 |
| 状态控制 | 发布 / 草稿 切换 |
| 置顶开关 | 是否置顶显示 |

**技术要点**:
- 动态路由 `/article/edit/:id?` 区分新建和编辑模式
- `el-upload` 组件实现封面上传预览
- 表单校验确保数据完整性

---

### 2.4 分类管理

**功能描述**: 以树形结构管理文章分类体系。

**核心能力**:
- **树形表格展示**: 使用 `el-table` 的 tree-props 实现多级分类嵌套展示
- **CRUD 操作**: 新增顶级分类、新增子分类、编辑、删除
- **排序**: 支持自定义排序值
- **关联统计**: 显示每个分类下的文章数量
- **弹窗交互**: 新增/编辑使用 `el-dialog` 弹窗表单

**适用场景**:
```
前端技术
├── Vue
├── React
└── TypeScript
后端开发
工程化
其他
```

---

### 2.5 标签管理

**功能描述**: 灵活的标签体系，辅助文章多维度归类。

**核心能力**:
- **Tag 云展示**: 使用 `el-tag` 组件以标签云形式展示所有标签
- **文章计数**: 每个标签旁显示关联文章数
- **快速操作**: 点击关闭图标即可删除标签
- **新增弹窗**: 输入名称快速创建新标签

**与分类的区别**:
- 分类：层级结构，一篇文只属一个主分类
- 标签：扁平结构，一篇文可关联多个标签

---

### 2.6 评论管理

**功能描述**: 对博客评论进行审核和管理。

**核心能力**:
- **评论列表**: 分页展示所有评论（ID、内容、所属文章、评论者、状态、时间）
- **状态筛选**: 待审核(黄) / 已通过(绿) / 已拒绝(红)
- **审核操作**: 一键通过待审核评论
- **删除操作**: 二次确认删除不当评论
- **内容搜索**: 按评论内容关键字搜索

**状态流转**:
```
用户发表 → [待审核] → [已通过] (前台可见)
                  └→ [已拒绝] (隐藏)
```

---

### 2.7 用户管理

**功能描述**: 管理系统注册用户信息。

**核心能力**:
- **用户列表**: 头像+用户名+邮箱组合展示
- **角色标识**: 管理员(红色Tag) / 普通用户(灰色Tag)
- **状态开关**: `el-switch` 一键启用/禁用账号
- **批量操作下拉菜单**: 修改角色、重置密码
- **搜索**: 按用户名或邮箱模糊搜索
- **分页**: 标准分页组件

**安全设计**:
- 重置密码需二次确认弹窗
- 禁用用户无法登录系统

---

### 2.8 系统设置

**功能描述**: 全局配置中心，分 Tab 管理不同维度设置。

#### 2.8.1 基本设置
- 站点名称 / 站点描述
- 站点 Logo 上传
- ICP 备案号

#### 2.8.2 评论设置
- 评论总开关
- 是否需要人工审核
- 敏感词配置（逗号分隔）

#### 2.8.3 上传设置
- 存储方式选择：本地 / 阿里云OSS / 腾讯云COS
- 最大文件大小限制
- 允许上传的图片格式（JPG/PNG/GIF/WebP）

## 3. 公共组件/布局

### 3.1 主布局 (MainLayout)

经典后台管理布局：
```
┌─────────────────────────────────────────────┐
│ Sidebar │        Navbar                     │
│  (侧边栏) ├─────────────────────────────────┤
│          │                                 │
│          │         Content Area             │
│          │      (路由视图 + 过渡动画)         │
│          │                                 │
└──────────┴─────────────────────────────────┘
```

**特性**:
- 侧边栏可折叠（220px ↔ 64px），宽度过渡动画 0.3s
- 深色主题侧边栏 (#304156)
- 页面切换淡入淡出过渡动画
- NProgress 顶部加载进度条

### 3.2 侧边栏 (Sidebar)

- Logo 区域（折叠时仅显示图标）
- 根据 router 配置动态生成菜单
- 支持 `el-sub-menu` 多级菜单展开
- 当前路由高亮（active-text-color）
- 图标使用 Element Plus Icons

### 3.3 顶部导航栏 (Navbar)

- 左侧：折叠按钮 + 面包屑导航
- 右侧：用户头像 + 用户名 + 下拉菜单（个人中心 / 退出登录）
- 退出登录二次确认

## 4. 技术亮点

| 特性 | 实现方式 |
|------|---------|
| **TypeScript 全覆盖** | 所有 .ts/.vue 文件均使用 TS 类型约束 |
| **按需引入** | unplugin-auto-import + unplugin-vue-components 自动导入 |
| **状态持久化** | pinia-plugin-persistedstate 自动缓存到 localStorage |
| **API 封装** | Axios 拦截器统一处理 Token 注入和错误响应 |
| **路由守卫** | 白名单机制 + Token 校验 + 动态标题 |
| **响应式图表** | ECharts 监听 resize 事件自适应容器 |
| **SCSS 变量** | 统一的设计令牌（颜色、间距、字号等） |
| **开发代理** | Vite dev proxy 解决跨域问题 |

## 5. 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器 (http://localhost:3000)
npm run dev

# 类型检查
npm run typecheck

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```
