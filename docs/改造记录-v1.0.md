## 三、布局改造

### 3.1 前台布局（BlogLayout）

```
┌─────────────────────────────────────────────────────┐
│  [左侧边栏: 260px]          │  [主内容区]              │
│  ┌──────────────────┐      │                          │
│  │ ✦ Vellastra      │      │  router-view             │
│  ├──────────────────┤      │                          │
│  │ [用户头像]        │      │  (首页/文章详情/写文章/   │
│  │ 用户名/角色      │      │   个人管理/个人中心等)     │
│  ├──────────────────┤      │                          │
│  │ 🏠 首页           │      │                          │
│  │ 📝 我的文章       │      │                          │
│  │ 📊 个人管理       │      │                          │
│  │ 👤 个人中心       │      │                          │
│  │ ⚙️ 后台管理       │      │                          │
│  │    (admin only)   │      │                          │
│  ├──────────────────┤      │                          │
│  │ [✍️ 写文章]       │      │                          │
│  │ [退出登录]        │      │                          │
│  │ [折叠按钮]        │      │                          │
│  └──────────────────┘      │                          │
└─────────────────────────────────────────────────────┘
```

- 左侧边栏可折叠（260px ↔ 72px）
- 背景为深空渐变 + 星星动画
- 卡片使用玻璃材质（backdrop-filter: blur）

### 3.2 后台管理布局（MainLayout）

- 左侧菜单，紫色高亮 + 透明背景
- 顶部导航栏：折叠按钮 + 面包屑 + 用户下拉

## 七、开发注意事项

- 所有新组件使用 `lang="scss"` 并引用 `variables.scss` 变量
- 前台页面统一使用 `BlogLayout.vue` 布局
- 管理员页面统一使用 `MainLayout.vue` 布局
- 紫色主题使用 `$primary-color: #7c3aed` 作为主色
- 玻璃卡片效果：`background: rgba(255,255,255,0.04); backdrop-filter: blur(12px)`
- 渐变文字：使用 `class="gradient-text"` 或 `background: linear-gradient(...) + -webkit-background-clip: text`
